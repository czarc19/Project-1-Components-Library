Table

Tables are a nice way to organize lots of data. We provide a few utility classes to help you style your table as easily as possible. In addition, to improve mobile experience, all tables on mobile-screen widths are centered automatically.

Table

An HTML table is defined with the "table"tag. Each table row is defined with the "tr" tag. A table header is defined with the "th" tag. By default, table headings are bold and centered. A table data/cell is defined with the "td" tag.
